# Portal_SEDU

Portal_SEDU é um site estático (HTML/CSS/JS) desenvolvido para o Portal da Secretaria de Educação (SEDU). Este repositório contém os arquivos do front-end do portal: a página principal (`index.html`), estilos em `css/` e scripts em `js/`.

## Descrição

O objetivo deste projeto é fornecer uma interface simples e responsiva para o portal, com navegação, conteúdo informativo e recursos front-end básicos.

## Estrutura do projeto

- `index.html` — página principal do portal
- `css/` — arquivos de estilo (ex.: `index.css`)
- `js/` — scripts JavaScript (ex.: `index.js`)
- `img/` — imagens utilizadas no site

## Como testar localmente

Opções rápidas para abrir o site localmente:

1. Abrir diretamente no navegador
- Clique duas vezes em `index.html` ou abra-o com o navegador de sua preferência.

2. Usar um servidor HTTP simples (recomendado para evitar problemas com CORS ao testar requisições locais):

- Com Python 3 (Windows PowerShell):

```powershell
Set-Location -LiteralPath "c:\Users\Karla Filgueiras\Desktop\Outras Paginas\Portal"
python -m http.server 8000
```

Depois abra `http://localhost:8000` no navegador.
